# 🎨 Text to Image Generator

This project is a simple **Text-to-Image Generator** web application built using **Streamlit** and **Hugging Face API**.

Users can enter a text prompt, and the application generates an image based on the input using an AI model.

---

## 🚀 Features

- Convert text into images using AI
- Simple and user-friendly interface
- Fast image generation using Hugging Face API
- Deployed on Streamlit Cloud

---

## 🛠 Technologies Used

- Python
- Streamlit
- Hugging Face API
- Requests Library
